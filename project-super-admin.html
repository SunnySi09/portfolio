Challenge
B2B diamond e-commerce involves highly complex parameters: multi-tiered pricing, custom certification variations, and diverse roles across stores. The existing administration workflow was fragmented, causing vendors and store operators to configure inventory pipelines manually on spreadsheets, leading to errors and delays.
Approach
1. Stakeholder Alignment & Mapping
Conducted product research alongside development leads to outline diamond configuration logic, API restrictions, and multi-tenant constraints.
2. Wireframing & Prototyping
Built low-fidelity grid wireframes in Figma to refine columns and filters, ensuring store managers could verify and edit pricing margins with two clicks.
3. Component Library Standardization
Designed structured UI component packages with strict design guidelines, color variables, and interactive states to facilitate faster front-end implementation.
Solution
We created a unified Admin Console featuring custom product publishers, bulk metadata updates, granular role authorization grids (RBAC), and automated pricing margins. A clean visual style hierarchy, standard hover feedback, and clear labels reduce data editing friction.
Results
43% reduction in catalog upload time
0% manual entry formula overwrite errors
