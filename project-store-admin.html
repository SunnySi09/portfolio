Challenge
Store administrators had to hop between different dashboard panels to handle basic store configurations, customer support roles, layout tweaks, and billing stats. This fragmented flow resulted in admin fatigue, user errors, and high onboard training costs for new tenants.
Approach
1. UX Flow Mapping
Mapped out the exact flows for store setup, user invite pipelines, and inventory sync alerts on FigJam to group settings logically.
2. Responsive UI Grid Wireframe
Designed a clean dashboard sidebar grid layout that shrinks into a top navigation bar on tablet and mobile viewports.
3. Role Access Tables (RBAC)
Created interactive permission selectors letting store admins tick toggle switches for individual feature access across role tiers.
Solution
We delivered a clean B2B|B2C Admin Dashboard with a standardized design language. By standardizing component grids, button paddings, status labels, and dashboard metrics cards, new users can easily navigate through catalog setups and role adjustments.
Results
30% decrease in support setup queries
95% task completion rate on first try
